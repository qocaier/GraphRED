Qocai GraphRED 2022

-------------------------

В программе GraphRED можно редактировать изображения, накладывать на них фильтры, рисовать и многое другое!

Для выбора изображения требуется перейти по меню: Файл >> Открыть. Принимаются только изображения форматов jpg, png
Сохранение: Файл >> Сохранить
Выход можно совершить не только нажатием на крестик: Параметры >> Выход

Отображаются слои в виде вкладок, где слева самые нижние, а справа - верхние. Для добавления слоя нажмите "+ Слой", для удаления - на крестик нужного слоя. Чтобы 
переименовать слой, необходимо дважды щелкнуть по нужному слою, ввести новое имя и нажать Enter. Для перемещения слоя вверх или вниз нажмите на него и, 
удерживая переместите его. Можно очистить слой, нажав на кнопку "Очистить слой"

Масштаб можно регулировать слайдером или задав нужное значение в процентах от 10 до 1000

Вашему вниманию представлено 4 фильтра:
# Ахроматизация - цвета приводятся в черно-белые
# Негатив - цвета становятся противоположными
# Размытие - происходит размытие изображения
# Анаглиф - создается изображение наподобие тех, которые видны в 3D очках, происходит сдвиг красной составляющей

Изюминкой моей программы является темная тема, которую можно включить через Параметры >> Темная тема
Также можно задавать изначальные настройки в settings.txt, где через пробел записываются размер кисти, прозрачность кисти, цвет кисти, масштаб, цветовая тема (w или b)
